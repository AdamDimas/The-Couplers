BAD CODE SMELL

Bad Code Smell merupakan istilah yang menandakan bahwa di suatu kode program mungkin terdapat suatu masalah. Kode yang dikatakan mengandung smell berarti ada yang salah, atau lebih tepatnya tidak seharusnya ada pada kode tersebut.

Meskipun code smell tidak mencegah program untuk berfungsi dengan baik, namun keberadaannya mengindikasikan adanya kelemahan desain yang dapat memperlambat proses pengembangan atau meningkatkan risiko munculnya bug di masa depan. Code smell juga dapat menjadi indikator dari faktor-faktor yang berkontribusi pada technical debt dalam pengembangan perangkat lunak.

Bad code smell bukan merupakan masalah yang langsung terlihat, tetapi merupakan sinyal adanya potensi masalah yang lebih dalam pada desain dan kualitas code. Mengidentifikasi dan mengatasi code smell sejak dini dapat membantu menjaga kualitas code dan menghindari masalah yang lebih besar di kemudian hari dalam proses pengembangan perangkat lunak.
